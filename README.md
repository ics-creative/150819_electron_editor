Electron Text Editor
==

ICS MEDIAの記事「[Electron入門 - ファイル保存可能なテキストエディターを自作しよう ](https://ics.media/entry/8401/)」で使用しているサンプルです。

詳しくはICS MEDIA記事をご覧ください。

## 使い方

Node.jsが入っているマシンでインストールする。

```bash
npm install
```

実行コマンドは以下。

```bash
npm run start
```
